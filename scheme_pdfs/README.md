# MyScheme Dataset

## Overview
This dataset contains structured information extracted from the Indian Government's **MyScheme** website ([myscheme.gov.in](https://www.myscheme.gov.in)). The data is collected from the sitemap and processed for accessibility, research, and analytical purposes.

## Data Type
723 PDF with Text inside each PDF (copyable) most of the schemes are here.

## Data Source
- Official Website: [MyScheme](https://www.myscheme.gov.in)
- Sitemap URL: [Sitemap](https://www.myscheme.gov.in/sitemap.xml)

## Dataset Contents
The dataset includes metadata and details about various government schemes listed on MyScheme. The extracted data may contain:

- **Scheme Name**: Title of the government scheme.
- **Description**: Brief information about the scheme.
- **Eligibility Criteria**: Who can apply for the scheme.
- **Benefits**: Incentives or support provided by the scheme.
- **Application Process**: Steps to apply for the scheme.
- **Official Link**: URL to the scheme's official webpage.

## Dataset Format
The dataset is provided in the following formats:
- **CSV**: Easy-to-use tabular data format.
- **JSON**: Structured data format for API and NLP applications.
- **Parquet**: Optimized format for big data processing.

This dataset can be used for:
